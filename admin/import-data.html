Petunjuk Import Data

  1. Unduh template JSON dengan menekan tombol "Download Template" di bawah
  2. Isi data produk sesuai format yang telah disediakan
  3. Pastikan semua field required diisi (type, model, spec, condition, user, pic, status, priceOriginal, priceAuction)
  4. Status yang tersedia: "Lelang", "Peremajaan", "Terjual", "Draft"
  5. Upload file JSON yang telah diisi menggunakan tombol "Pilih File"
  6. Review data sebelum melakukan import dengan menekan "Preview Data"
